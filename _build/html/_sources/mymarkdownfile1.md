## What is a time series?

```{Tip}
A time series is a realization of a random variable indexed by time.
```
    Definition:
    
    A time series is a realization of a random variable indexed by time.

A time series can be denoted as

$$
X_t = \{ x_1, x_2, \ldots, x_T \}
$$

Our task is to analyze the above stochastic process.
